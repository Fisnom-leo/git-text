was ist Git

Git ist ein Open-Source-Tool welches für die Versionskontrolle von Software verwendet wird. Es wurde 2005 von Linus Torvalds ins Leben gerufen wurde. Das Wort “Git” kommt aus der britischen Umgangssprache und bedeutet übersetzt soviel wie “Blödmann”.
